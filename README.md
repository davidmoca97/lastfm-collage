# lastfm-collage
Collage generator for Last.fm made with Go

![App running in the browser](https://i.imgur.com/EEWumOs.png)
![Collage example](https://i.imgur.com/nd1vQoK.png)

### How to run it?

1. Clone the repo
2. Open the terminal and go to the repo folder
3. Open `config/config.go` and add your own `LastFMApiKey`
4. Run `go run main.go`
5. Open the browser in http://localhost:999
